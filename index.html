Hyperlink Example

A hyperlink is indicated by the <a> tag like THIS ONE LINKS TO BAIDU.

The 'herf' property defines what resource the hyperlink refers to. It can be a web address like "http://www.baidu.com" in the link above. And it can also be any file in the same folder as this html file like THIS ONE LINKED TO AN IMAGE FILE.

You need to understand how the url works to make sure the path of file you link with the <a> tag. Basicly, there are two types, the absolute path and the relative path.

The absolute path strictly defines where the file is on your machine or the Internet. So if the file itself is moved to another place, or the html page works on a different machine, the absolute link may not function properly. THIS ABSOLUTE LINK is linked to a file on my Mac as a result you can't see the file when you click the link. And if it is linked to a file on the Internet, make sure the file will always be there for your visitors with time goes by.

The relative link takes the folder of this html file as a starting point to look for files. The '.' means where the html file locates. The '/' means enter the folder after it or find the exact file name. THIS LINK demonstrates how to link a file in the subfolder of the htmls file folder. Because it uses a relative path, it works well on any machine as long as you copy and paste the entire file folder. Another useful trick is '..' which means going to the file folder in the upper level of where the html file is.

If you would like the browser to OPEN A NEW WINDOW to see the link, set the 'target' property to '_blank' in the <a> tag.

Sometime you need to let people send emails to you. And link an EMAIL ADDRESS to a hyperlink on your page may be convenient. It take the 'mailto:yourusername@mailserver.com' in the 'href'. After being clicked, it will open the default email software on your machine with the email address filled in the 'receiver' blanket, typically the Mail App on Mac and the Outlook on Windows.
